# Australian Weather Analytics Web Application

A Python and SQLite web application for exploring and analysing historical Australian weather station and climate data through an interactive web interface.

## Features

* Search historical weather stations by state and location
* Explore historical climate metrics including temperature, rainfall, evaporation, and sunshine
* Compare weather stations across different time periods
* Analyse similarities between historical climate trends
* Display results through a clean HTML/CSS interface

## Technologies

* Python
* SQLite
* SQL
* HTML
* CSS

## Dataset

This project uses historical Australian weather station and climate data stored in an SQLite database. The application allows users to query, filter, compare, and analyse past climate observations across multiple locations and time periods.

## How to Run

1. Clone the repository.
2. Install Python 3.
3. Ensure the SQLite database is located in the `database` folder.
4. Run:

```bash
python demo3.py
```

5. Open your browser and visit:

```
http://localhost
```
